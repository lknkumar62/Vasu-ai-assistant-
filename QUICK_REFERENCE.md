# 🎯 Vasu AI - Quick Reference Card

## 🚀 Getting Started (30 seconds)
```bash
cd vasu-ai-assistant
npm install
cd backend && npm install && cd ..
cp .env.example .env
# Add your OPENAI_API_KEY to .env
npm run dev
```

## 🗣️ Essential Voice Commands

### Device Control
| Command | Action |
|---------|--------|
| "Vasu, open [app]" | Opens any app |
| "Vasu, call [name]" | Makes phone call |
| "Vasu, WiFi on/off" | Toggles WiFi |
| "Vasu, flashlight on" | Turns on torch |
| "Vasu, volume [0-100]" | Sets volume |

### Communication
| Command | Action |
|---------|--------|
| "Vasu, WhatsApp [name]: [msg]" | Sends WhatsApp |
| "Vasu, SMS [number]: [msg]" | Sends SMS |
| "Vasu, email [addr]: [subj]" | Sends email |

### Media
| Command | Action |
|---------|--------|
| "Vasu, play [song/artist]" | Plays music |
| "Vasu, pause/stop" | Stops music |
| "Vasu, next/previous" | Changes track |

### AI Coding
| Command | Action |
|---------|--------|
| "Vasu, create React app" | Generates React code |
| "Vasu, write Python script" | Generates Python code |
| "Vasu, deploy to Vercel" | Deploys project |

### Learning
| Command | Action |
|---------|--------|
| "Vasu, teach me [topic]" | Starts learning |
| "Vasu, quiz me" | Starts quiz |
| "Vasu, revision" | Reviews due items |

### Automation
| Command | Action |
|---------|--------|
| "Vasu, Good Morning" | Runs morning routine |
| "Vasu, schedule [task] at [time]" | Sets reminder |
| "Vasu, create routine [name]" | Creates shortcut |

## ⌨️ Keyboard Shortcuts (Web/Desktop)

| Shortcut | Action |
|----------|--------|
| `Space` | Toggle voice listening |
| `Esc` | Stop speaking/listening |
| `Ctrl + M` | Open mic |
| `Ctrl + S` | Open settings |
| `Ctrl + H` | Show help |

## 🔧 Troubleshooting Quick Fixes

| Problem | Solution |
|---------|----------|
| App won't start | `expo start -c` |
| Backend error | Check `.env` API keys |
| Voice not working | Grant mic permissions |
| Commands fail | Check internet connection |
| Build fails | Clear cache: `npm cache clean` |
